# PCAP Traffic Analyzer
Система для глубокого анализа сетевого трафика, который превращает сырые .pcap файлы в понятную аналитику и отчеты об угрозах.
Представляет из себя web-приложение, которое автоматически разбирает пакеты на потоки (flows) и пропускает их через ряд специализированных детекторов для выявления аномалий.
Полученная информация отображается на дашборде.

## Основной процесс работы
1. Загрузка .pcap файла через веб-интерфейс.
2. Декодирование пакетов на бэкенде.
3. Группировка пакетов в сессии (Flows) на основе IP-адресов, портов и протоколов.
4. Анализ каждого потока набором алгоритмических детекторов.
5. Сохранение результатов и метаданных.
6. Отправка данных на фронтенд и отображение на интерактивных дашбордах.

## Детекторы угроз
- DDoS Detector
- Перегрузка
- Сканирование портов
- Деятельность червей

## Технологический стек
Backend: `Golang`
Database: `PostgreSQL`
Frontend: `React`
